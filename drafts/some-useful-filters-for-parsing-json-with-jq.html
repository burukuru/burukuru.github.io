Some useful filters for parsing JSON with jq

Posted on Wed 06 May 2015 in Uncategorized

I've been told jq is the new cool JSON parser. As I used to use the 'json' command that came from npm, I had to adapt my scripts slightly.

jq -r .

Re-arrange the output:

jq -r '.[] | {name: .name, dob: .dob} | join (" ")'

jq -r '[.options[] | select(.name=="blah")'
